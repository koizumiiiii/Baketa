# Baketa

**ゲーム翻訳オーバーレイアプリケーション**

Baketaは、ゲーム画面上のテキストをリアルタイムで翻訳し、透過オーバーレイとして表示するWindowsアプリケーションです。

---

## 動作要件

### 必須環境
- **OS**: Windows 10/11 (64-bit)
- **CPU**: Intel Core i5 / AMD Ryzen 5 以上
- **メモリ**: 8GB以上（16GB推奨）
- **ストレージ**: 5GB以上の空き容量（翻訳モデル用）
- **インターネット接続**: 初回起動時に必要（翻訳モデルのダウンロード）

### 推奨環境（GPU使用時）
- **GPU**: NVIDIA GTX 1060以上（CUDA対応）
- **VRAM**: 4GB以上

> **Note**: GPUがなくてもCPUモードで動作しますが、翻訳速度が遅くなります。

---

## インストール方法

1. ダウンロードしたZIPファイルを任意のフォルダに展開します
2. 展開したフォルダ内の `Baketa.exe` をダブルクリックして起動します

**注意**: 初回起動時に翻訳モデル（約2GB）が自動的にダウンロードされます。

---

## 翻訳モード

Baketaには2つの翻訳モードがあります：

### Live翻訳（自動翻訳）

ゲーム画面を継続的に監視し、テキストが変化すると自動で翻訳します。

- **Start** ボタンで開始
- **Stop** ボタンで停止
- テキストが多いRPGやアドベンチャーゲームに最適

### Shot翻訳（手動翻訳）

ボタンを押した瞬間の画面を1回だけ翻訳します。

- **Shot** ボタンで実行
- メニュー画面やステータス画面など、変化の少ない場面で便利
- 必要な時だけ翻訳するため、リソース消費を抑えられます

---

## 基本的な使い方

### 1. ゲームを起動
翻訳したいゲームを通常通り起動します。

### 2. Baketaを起動
`Baketa.exe` を起動します。

### 3. ゲームウィンドウを選択
メイン画面でゲームウィンドウを選択します。

### 4. 翻訳を開始
- **自動翻訳**: **Start** ボタンをクリック
- **手動翻訳**: **Shot** ボタンをクリック

### 5. 翻訳を停止
**Stop** ボタンをクリックして翻訳を停止します。

## トラブルシューティング

### アプリが起動しない

- **原因**: .NET Runtimeがインストールされていない可能性があります
- **解決策**: [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0) をインストールしてください

### モデルのダウンロードに失敗する

- **原因**: ネットワーク接続の問題
- **解決策**:
  1. インターネット接続を確認してください
  2. ファイアウォールやセキュリティソフトがブロックしていないか確認してください
  3. しばらく待ってから再度起動してください

### 翻訳結果が表示されない

- **原因**: ゲームウィンドウが正しく選択されていない可能性があります
- **解決策**:
  1. ゲームウィンドウを再選択してください
  2. ゲームをウィンドウモードまたはボーダーレスウィンドウモードで実行してください

### オーバーレイが見づらい

- **解決策**: 設定画面でフォントサイズを調整してください

### 翻訳が遅い

- **原因**: CPUモードで動作している可能性があります
- **解決策**: NVIDIA GPUをお持ちの場合、CUDAドライバーが正しくインストールされているか確認してください

---

## ログファイルの場所

問題が発生した場合、以下の場所にあるログファイルをご確認ください：

```
%USERPROFILE%\.baketa\settings\logs\
```

例: `C:\Users\<ユーザー名>\.baketa\settings\logs\`
