FAQ

Is Baketa really free?
Yes! Baketa is free and open source. Basic features including local OCR and translation are completely free. EX Mode, a high-accuracy translation feature, is optionally available for supporters.
Does it work with any game?
Baketa works with most windowed and borderless-windowed games on Windows. Full-screen exclusive mode may require switching to borderless windowed mode.
Does it affect game performance?
Baketa is designed to be lightweight and most users report no noticeable slowdown. However, games with heavy memory or GPU usage may experience some impact.
What is the difference between Live and Shot translation?
Live translation continuously monitors the screen and translates text as it appears. Shot translation captures and translates a single frame on demand — useful for menus or static text.
Is an internet connection required?
No. OCR and local translation work fully offline. EX Mode requires an internet connection but is optional.
